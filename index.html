¿Qué es un Dato?

Un dato es una representación simbólica de una unidad básica de información que, por sí sola, no tiene un significado completo.

¿Qué es información?

La información es un conjunto de datos procesados y organizados de manera que adquieren significado y relevancia para el usuario.

¿Diferencia entre dato e información?

Un dato es un hecho bruto y sin procesar, mientras que la información es el resultado de procesar y organizar esos datos para proporcionar significado y contexto.

Tipos de datos:

Enteros (int)
Flotantes (float, double)
Caracteres (char)
Cadenas de texto (string)
Booleanos (boolean)
Fechas (date)

¿Qué es un modelado de Datos?

El modelado de datos es el proceso de crear una estructura de datos para representar las entidades y sus relaciones dentro de un sistema de información.

Tipos de bases de Datos, según:

Variabilidad:

Estáticas: Los datos no cambian.
Dinámicas: Los datos cambian con el tiempo.

Contenido:

Documentales: Gestionan documentos.
Bibliográficas: Gestionan referencias bibliográficas.
De multimedios: Gestionan imágenes, videos, etc.

Modelo de base de datos:

Relacional: Utiliza tablas.
NoSQL: Almacena datos en formatos como documentos, clave-valor, gráficos.
Jerárquica: Organiza datos en una estructura de árbol.
Red: Utiliza un grafo para representar relaciones.

¿Qué es el modelo relacional?

El modelo relacional es un tipo de base de datos que organiza los datos en tablas, las cuales están relacionadas entre sí mediante claves primarias y foráneas.

Describa los elementos del Diagrama entidad Relacional:

Entidad: Representa un objeto del mundo real con existencia independiente.
Atributo: Propiedad o característica de una entidad.
Relación: Asociación entre entidades.
Llave primaria: Identificador único para una entidad.
Llave foránea: Campo que crea una relación entre dos tablas.

¿Qué es una entidad?

Una entidad es un objeto del mundo real que se puede identificar de forma única y sobre el cual se desea almacenar información.

¿Qué es un atributo y sus tipos?

Un atributo es una propiedad o característica de una entidad. Tipos de atributos incluyen:
Simples y compuestos.
Monovalentes y multivalentes.
Derivados.

¿Qué es una relación?

Una relación es una asociación entre dos o más entidades.

¿Qué son las llaves primarias, foránea y sus características?

Llave primaria: Identificador único de una entidad. Debe ser única, no nula y no cambiable.
Llave foránea: Campo en una tabla que se refiere a la llave primaria de otra tabla, estableciendo una relación entre ambas.

¿Qué son las reglas del Negocio?

Las reglas del negocio son directrices y restricciones que determinan cómo se deben gestionar los datos y procesos dentro de una organización.

¿Qué es la cardinalidad o mapeo?

La cardinalidad define el número de instancias de una entidad que pueden asociarse con instancias de otra entidad en una relación.
Se puede clasificar en uno a uno, uno a muchos, y muchos a muchos.
